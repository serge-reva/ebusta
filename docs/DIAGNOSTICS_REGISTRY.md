# DIAGNOSTICS_REGISTRY.md

## Реестр диагностических кодов (DSL-парсер)

> **Важно**: в текущей реализации диагностика возвращается в поле `error_msg` строкой без стандартизированных кодов.  
> Данный документ описывает **целевое состояние** и должен быть реализован в будущих версиях.

| Код | Значение | Пример | Severity |
|-----|----------|--------|----------|
| `DSL_UNSPECIFIED` (0) | Неизвестная ошибка | fallback | error |
| `DSL_EMPTY_INPUT` (1) | Пустой запрос после тримминга | `""` | error |
| `DSL_SYNTAX` (2) | Синтаксическая ошибка | `author:` | error |
| `DSL_UNKNOWN_FIELD` (3) | Неизвестное поле в запросе | `publisher:oreilly` | warning |
| `DSL_UNBALANCED_PARENS` (4) | Несбалансированные скобки | `(author:king` | error |

### Правила
- Коды **иммутабельны** – новые добавляются в конец.
- При генерации из кода (например, через комментарии в `dsl-scala`) должен создаваться файл `diagnostics.gen.md`.
