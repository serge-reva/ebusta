# GLOSSARY.md

## Глоссарий терминов

| Термин | Определение |
|--------|-------------|
| **DSL** | Domain-Specific Language – язык запросов, используемый в ebusta (например, `author:king AND title:it`). |
| **AST** | Abstract Syntax Tree – внутреннее представление разобранного DSL-запроса. |
| **TraceID** | Уникальный идентификатор запроса для сквозной трассировки через все сервисы. |
| **Gateway** | Единая точка входа для внешних клиентов (кроме CLI), выполняющая маршрутизацию, валидацию и rate limiting. |
| **Orchestrator** | Сервис, координирующий поисковый пайплайн: DSL → Query Builder → DataManager. |
| **DataManager** | Сервис, выполняющий запросы к OpenSearch. |
| **Downloader** | HTTP-сервис для скачивания файлов книг. |
| **PlasmaNode** | In-memory кэш для файлов книг. |
| **TierNode** | Дисковый кэш. |
| **ArchiveNode** | Узел, работающий напрямую с zip-архивами. |
| **StorageNode** | Общий gRPC-интерфейс для узлов хранения. |
| **OpenSearch** | Поисковый движок, хранящий индекс книг. |
| **Rate limiting** | Ограничение частоты запросов (реализовано в пакете `edge`). |
| **Errutil** | Пакет для унифицированной обработки ошибок с кодами и TraceID. |
