# UI_MVP_0_01_CONTRACT.md

## Контракт веб-интерфейса (web-frontend)

### Поиск
- Форма с полем ввода и кнопкой "Search".
- При отправке выполняется GET-запрос к `/search?q=<query>&page=<page>`.
- Результаты отображаются в виде таблицы с колонками: Title, Authors, ID, File.
- При клике на строку происходит переход на `/download/{id}`.

### Пагинация
- Параметр `page` в URL.
- Отображаются номера страниц с навигацией (первая, предыдущая, следующая, последняя).

### TraceID
- На странице не отображается (но может быть добавлен для отладки).
- В запросах не передаётся – генерируется на стороне gateway и возвращается в заголовке `X-Trace-Id`.

### Обработка ошибок
- При ошибке поиска показывается сообщение "Ошибка поиска" и, если доступен, TraceID.
- При ошибке скачивания – JSON-ответ с ошибкой (для API-клиентов) либо перенаправление на страницу с ошибкой (для пользователей).

### Технологии
- Чистый HTML + CSS (без фреймворков).
- JavaScript минимален (только для обработки кликов по строкам, если требуется).
